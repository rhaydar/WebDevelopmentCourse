Why This Website is Cooler Than Yours

While the rest of the world is stuck in the last decade, this site is built using HTML5.

HTML5 has a lot of advantages over prior versions of HTML. Here are just a few:

  • Cleaner code
  • Better for mobile devices
  • Improved audio and video support
  • Much better design of web forms
  • Accessibility improvements for disabled visitors

In the coming years, most major websites will built using HTML5. But I didn't want to wait, so I jumped right in!
